== README

Wingman: A Date Planning App

by: Charlene Estiva, David Simons, Alan Van Pelt and Matthew Lopez

Wingman is designed to make planning a date in San Francisco a snap.
Plug in events you're interested in, select a restaurant near that location,
then save those details to your date itinerary. You can share the itinerary
with your intended date if you wish. It's that easy.
